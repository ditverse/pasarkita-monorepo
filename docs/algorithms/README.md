# Dokumentasi Algoritma PasarKita

Dokumen ini menjelaskan dua algoritma utama yang digunakan dalam aplikasi
marketplace PasarKita, disertai referensi kode, cara kerja, dan langkah
verifikasi.

| Algoritma | Dokumen | Penerapan Utama |
|---|---|---|
| Greedy | [greedy-algorithm.md](./greedy-algorithm.md) | Rekomendasi produk, fee marketplace, sorting terlaris |
| String Matching | [string-matching-algorithm.md](./string-matching-algorithm.md) | Pencarian produk, pencarian pesanan |

## Cara Membaca Dokumentasi Ini

Setiap dokumen algoritma terdiri dari bagian-bagian berikut:

1. **Definisi** — Penjelasan singkat tentang apa itu algoritma tersebut.
2. **Penerapan di PasarKita** — Di bagian mana algoritma dipakai di aplikasi ini.
3. **Pseudocode** — Langkah-langkah algoritma dalam bahasa manusia.
4. **Kode Aktual** — Potongan kode nyata dari proyek ini.
5. **Cara Menguji** — Langkah konkret untuk membuktikan algoritma bekerja.
6. **Analisis Kompleksitas** — Big-O dari algoritma tersebut.
